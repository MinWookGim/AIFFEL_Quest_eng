# MQ02 재발표 (런케이션 해커톤 본선) — 추가 실험 기록

기수 발표에서 1등을 해서 런케이션 해커톤 본선에 진출했다 (2026-07-16, 발표 10분 + 질의 5분).
본선은 다른 기수들과 외부 손님들까지 듣는 자리라, 발표 자료를 처음 듣는 사람 기준으로 다시 짜면서
1차 발표 때 비어 있던 측정 하나를 이번에 직접 메웠다. 그 기록이다.

## 왜 이 실험을 했나 — 1차 때 대답할 수 없던 질문

1차 발표에서 앙상블(5모델 WBF)에 대해 이렇게 말했다.

- damage recall 0.856 -> 0.944 (자체 채점기, test 301장, conf 0.05)
- 대신 precision 0.394 -> 0.150 으로 하락

그런데 "그러면 앙상블의 **헛경보율**(깨끗한 이미지에서 잘못 경보를 울리는 비율)은 얼마냐"는 질문에는
답할 숫자가 없었다. 헛경보 30% -> 17%라는 숫자는 **단일 모델**을 정상 이미지 200장에 돌려 잰 것이고,
앙상블은 test 301장에서만 쟀기 때문이다. 서로 다른 데이터의 숫자라 붙여 말할 수 없었고,
그래서 1차 때는 "정상 주입과 앙상블로 두 마리 토끼를 잡았다" 같은 말을 하지 않기로 했었다.
측정하지 않은 주장은 하지 않는 게 원칙이라서다.

재발표를 준비하면서, 이 구멍을 실험으로 메웠다.

## 방법 — 단일 모델 측정과 완전히 같은 조건

비교가 성립하려면 잣대를 바꾸면 안 된다. 그래서 전부 기존 그대로 맞췄다.

- 데이터: 단일 모델 헛경보 측정 때 쓴 것과 **같은 정상(라벨 없는) 이미지 200장**
- 헛경보 정의: 한 이미지에 예측 박스가 1개라도 뜨면 그 이미지는 헛경보 (기존과 동일)
- 앙상블: 최종 발표본과 동일 구성 (YOLO 4종 norm 주입 모델 + rtdetr 가중치 0.5, WBF IoU 0.55)
- 문턱 2개를 병기: conf 0.25 (단일 모델을 쟀던 일반 문턱) / conf 0.05 (앙상블 recall 0.944가 나온 스크리닝 운영점)
- 검증 장치: 같은 코드에서 단일 챔피언(yolov8s_n1500)@0.25 가 **1차 실측(34/200 = 17.0%)을 정확히
  재현하는지** 먼저 확인했다. 재현됐다. 그래서 나머지 숫자를 믿는다.

## 결과

| 방법 | conf 0.25 (일반 문턱) | conf 0.05 (스크리닝 운영점) |
|---|---|---|
| 단일 챔피언 (yolov8s_n1500) | 17.0% (34/200) | 46.5% (93/200) |
| 앙상블 ENS_YOLO (4종) | **12.5%** (25/200) | 44.5% (89/200) |
| 앙상블 ENS_ALL (최종, +rtdetr) | **15.5%** (31/200) | **62.0%** (124/200) |

## 해석

1. **같은 문턱(0.25)에서 앙상블의 헛경보는 늘지 않았다. 오히려 소폭 줄었다(17.0% -> 15.5%).**
   이유를 생각해보면, WBF는 겹치는 박스의 점수를 모델들의 합의로 가중평균한다. 한 모델만 잘못 본
   헛박스는 다른 모델의 지지를 못 받아 점수가 희석되고, 문턱을 못 넘는 것이다. rtdetr 없이
   YOLO 4종만 합치면 12.5%까지 내려간다 (rtdetr을 넣으면 후보가 늘어 15.5%가 되는 대신 recall이 최고가 된다).
2. **스크리닝 문턱(0.05)에서는 46.5% -> 62.0%로 늘어난다.** recall 0.944는 후보를 넓게 쓸어담는
   운영점에서 나온 숫자이고, 그 대가는 사람이 확인해야 할 이미지가 느는 것이다. 우리 용도가
   사람이 최종 확인하는 스크리닝이라 받아들인 거래다.
3. 그래도 **"두 마리 토끼"라고는 말하지 않기로 했다.** recall 0.944(문턱 0.05)와 헛경보 15.5%(문턱 0.25)는
   서로 다른 운영점의 숫자라, 한 문장에 붙이면 존재하지 않는 운영점을 주장하게 된다.
   운영점별로 끊어 말하는 게 정확하다.

## 한계 (정직하게)

- 평가용 정상 200장과 학습에 주입한 정상 이미지가 **일부 겹친다** (주입은 seed 셔플 샘플이라
  1500장 기준 약 29장 겹침 추정). 1차의 단일 모델 측정도 같은 조건이었으므로
  **단일 vs 앙상블의 상대 비교는 공정**하지만, 헛경보율 절대값은 실제보다 낙관적일 수 있다.
- conf 0.50에서의 헛경보율은 재지 않았다 (recall 비교에서 이미 단일이 역전하는 문턱이라 실익이 없었다).
